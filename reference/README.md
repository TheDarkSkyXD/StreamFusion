# Reference

This folder contains reference materials for the StreamFusion project.

## What Belongs Here

- **API references** — external API docs for Kick, Twitch, and other integrations
- **Architecture diagrams** — system design and data flow diagrams
- **External documentation** — links or snapshots of third-party docs relevant to the project
- **Protocol notes** — WebSocket message formats, auth flows, rate limits, etc.
- **Design specs** — UI/UX references, style guides, component inventories
